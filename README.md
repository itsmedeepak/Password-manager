

# Password Manager



## Project Overview 💁

> It's a command line dekstop application, which helps users to generate strong random password and manage their password in local database.

## Quick start 🙌🏻

Several quick start options are available:

- [Download the latest release](https://github.com/itsmedeepak/Password-manager) as ZIP file
- Or clone the repo: `https://github.com/itsmedeepak/Password-manager.git`


## Dependencies ✍🏼✍🏼

- Install Python3 (latest release): `sudo apt install python3.9`
- Install tkinter module: `pip install tkinter`
- Install JSON module: `pip install json`
- Install Random: `pip install random`
- Install Pyperclicp module: `pip install pyperclip`


### Download all dependencies in a single command 🤷🏼‍♂️ 🤷🏼‍♂️ 
`pip install tkinter json random pyperclip`



## Get Started 🏆🏆
- Install all dependencies in your local pc.
- Clone the project into your local computer.
- Open clone folder in your terminal.
- For launching the application type `python3 main.py`
- A pop up window will open ie password manager application.
